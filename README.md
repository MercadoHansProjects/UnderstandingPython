# Understanding Python

A collection of small Python projects I built while learning the language and getting comfortable with core programming concepts.

## Projects

| File | Description |
|------|-------------|
| `Calculator.py` | Basic calculator for arithmetic operations |
| `Expense Tracker.py` | Tracks and logs expenses (see `expenses.csv`) |
| `GuessTheNumber.py` | Number guessing game |
| `Rock Paper Scissors.py` | Classic rock-paper-scissors game vs. the computer |
| `Magicball.py` | Magic 8-ball style random answer generator |
| `Number Generator.py` | Generates random numbers |
| `Password Generator.py` | Generates random secure passwords |
| `scanner.py` | Network port scanner |

## What I Learned

- Python syntax and control flow (loops, conditionals, functions)
- Working with user input and basic I/O
- File handling (CSV)
- Randomization and simple game logic
- Basic networking concepts (port scanning)

## Getting Started

Clone the repo and run any script with Python 3:

```bash
git clone git@github.com:MercadoHansProjects/UnderstandingPython.git
cd UnderstandingPython
python "script_name.py"
```
